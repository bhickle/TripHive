What's different from v2 (oval stamp): Instead of just the stamp, each card is now a passport page — cream/parchment background with a faint repeating "TRIPCOORD ✦" watermark, a spine-shadow on the left edge (like the page is bound into a book), and a country code chip at the top ("ITA", "JPN", "GRC"…). The actual stamp is pressed onto the page, rotated slightly, in vibe-coloured ink.

What stays distinct from badges: Badges keep their clean white-card-with-bright-emoji look. Passport pages are warm cream with monospace + watermark — completely different palette, no risk of confusion.

Implementation cost: Slightly heavier than v2 (~50 lines of CSS, one inline SVG watermark) but still pure CSS and no new image assets. The country code is derived from stamp.country (already in the data shape, just truncated to a 3-letter code).
